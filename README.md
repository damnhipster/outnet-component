# outnet-component

Create and share web components across the Outnet.

##Installation
```
npm install -g yo
npm install -g generator-outnet-component
```

##Usage
```
yo outnet-component your-component-name
```

##Options
```
-- react
-- polymer
-- vue
-- vanilla
```
